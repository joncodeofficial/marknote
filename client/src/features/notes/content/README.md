# Welcome to markNote

markNote is a personal markdown notebook focused on fast writing, clean reading, and simple organization.

## What you can do here

- Create notes and keep them organized from the sidebar
- Write in Markdown with a live editing experience
- Preview your content instantly as formatted text
- Import `.md` files from your device
- Rename, reorder, search, and delete notes quickly

## How it works

Use the **Editor** tab to write or update your note.

Use the **Preview** tab to check how your markdown will look once formatted.

Use the sidebar to move between notes, create new ones, or manage the ones you already have.

## Tips

- Right click a note to rename it
- Use the copy button in **Editor** or **Preview** to copy the current markdown
- Drop a markdown file when creating a note to import its content directly

## Start here

Create a new note from the sidebar and begin writing.
